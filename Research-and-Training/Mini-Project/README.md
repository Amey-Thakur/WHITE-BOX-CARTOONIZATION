# Mini-Project

Original training implementation and codebase for the White-Box Cartoonization research project.

## Overview

This directory contains the training code, datasets, demo files, and project documentation from the 6th Semester Mini-Project (Batch of 2022).

## Directory Structure

```
Mini-Project/
│
├── WBC/                               # Main implementation
│   ├── train_code/                    # GAN training code
│   ├── test_code/                     # Inference scripts
│   ├── images/                        # Sample images
│   ├── paper/                         # Research paper files
│   ├── index.html                     # Web interface
│   ├── STEPS.txt                      # Setup instructions
│   └── README.md                      # Detailed documentation
│
├── Demo/                              # Demonstration files
│   └── Sample outputs and examples
│
├── Files/                             # Supporting resources
│   └── Additional project files
│
├── Group - B11 [Amey, Mega & Hasan]/ # Group submissions
│   └── Project deliverables
│
└── README.md                          # This file
```
